HTML Introduction 📌

HTML is the standard markup language, It looks like a skeleton for a website.

A simple HTML structure


<!DOCTYPE html>
    <html>
    <head>
        <title>Page title</title>
    </head>
    <body>
        <h1>My first heading</h1>
        <p>My first paragraph</p>
    </body>
    </html>

This is our code preview

Let's talk about previous example

  • <!DOCTYPE html> This is defined, This website create with HTML5
  • <html> This is a root element
  • <head> This tag is used for add website metadata and stylesheet.
  • <title> This tag is used for add website title.
  • <body> We use this tag to add visible contents. like image, paragraph, hyperlink, lists, tables, etc
  • <h1> Element defines a large heading
  • <p> Element defines a paragraph

Let's talk about How HTML Work & How we write.

Okay, Html stands for a hypertext markup language.When we work with HTML, then we really need to keep in mind that every HTML element needs an opening & closing bracket. But there is some exception there like [<br>,<hr>,<img>,<link>] etc. This is a self-closing tag.

  • We need to start with < this. and then we need to place your element argument, Then this tag end with > let's write it together <html> This is an opening tag. Now it's time to close your tag using / this. Now the question is how ? So after first bracket we need to add / This. </html> This is a final closing tag.

Let's talk about HTML history

  1. 1989 - Tim Berners-Lee invented www
  2. 1991 - Tim Berners-Lee invented HTML
  3. 1995 - HTML Working Group defined HTML 2.0
  4. 1997 - W3C Recommendation: HTML 3.2
  5. 1999 - W3C Recommendation: HTML 4.01
  6. 2000 - W3C Recommendation: XHTML 1.0
  7. 2008 - WHATWG HTML5 First Public Draft
  8. 2016 - W3C Candidate Recommendation: HTML 5.1
